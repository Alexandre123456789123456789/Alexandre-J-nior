Campo de Senha : type="password"

Campo para escolher cor: type=color

Campo de data : type="date"

Campo de hora : type="time"

Campo para escolher data e hora : type="datetime-local"

Campo para escolher mês : type="month"

Campo para escolher a semana : type="week"

Campo para receber endereço de e-mail : type="email"

Imagem como botão de submissão : type="image"

A imagem funciona como um botão e irá executar a função definida no formulário.

Campo para enviar um único arquivo DOC: type="file"

Campo para enviar vários arquivos PDF ou de imagens: type="file"

Campo de entrada escondido - neste exemplo o valor é igual 321: type="hidden"

Embora o valor não seja exibido para o usuário no conteúdo da página, ele fica visível (e pode ser editado) usando as ferramentas de desenvolvedor de qualquer navegador ou a funcionalidade "Exibir código-fonte". Não use entradas ocultas como forma de segurança!

Campo para receber números inteiros: type="number"

Campo para receber números inteiros de 0 a 10: type="number"

Campo para receber números com casas decimais: type="number"

Campo de faixa de valores de 0 a 10 com saltos de 2 a 2: type="range"

Campo de pesquisa: type="search"

Campo para entrada de número de telefone: type="tel"

Campo para entrada de URL: type="url"

Prenchimento mínimo: http:dominio (na prática pode ser qualquer texto, somente verifica se tem o sinal:)

Botão submit com imagem: type="submit"

O texto é opcional.

Lista de opções com agrupamento: select, optgroup e option
